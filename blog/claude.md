## módulo de admin

 este modulo é responsavel por :

 - Mostar a lista de posts do blog a partir do slug do blog
 - e mostrar o conteudo de um post a partir do blog e do slug do post

## instruções

- Gerar código sem comentários
- Não utilizar JSON nem padrão de API
- gerar os teste unitários
- corrigir erros de arquitetura
- fazer um código limpo e se possível reutiliazando classes
- a conexão com o banco de dados está racunhada no /database/database.php
